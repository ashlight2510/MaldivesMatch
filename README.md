# 🏝️ 몰디브 매치 (Maldives Match)

나의 몰디브 여행 성향을 찾아주는 전문 성향 분석 테스트

## ✨ 주요 기능

- **18개의 전문 질문**으로 정밀한 여행 성향 분석
- **8가지 세분화된 성향 타입** 제공
- **6각형 레이더 차트**로 시각화
- **실제 리조트 추천** - 각 성향에 맞는 구체적인 리조트 이름 제공
- **반응형 디자인** - 모바일/데스크톱 모두 지원

## 🎯 8가지 성향 타입

1. **👑 럭셔리 힐링형** - 최고급 시설과 서비스 중시
   - 추천: 소네바, 식스센스, 벨라사루, 조아리 등

2. **🐠 수중 탐험가형** - 하우스 리프와 다이빙 중시
   - 추천: 릴리비치, 에라이두, 말라히니코두 등

3. **💙 라군 낭만형** - 인스타 감성과 예쁜 바다 색깔
   - 추천: 밀라이두, 피넥쉬, 칸두마 등

4. **💰 가성비 실속형** - 합리적 가격의 알찬 여행
   - 추천: 아다란클럽란날리, 메루, 비얏이 등

5. **🍽️ 미식 올인형** - 올인클루시브와 다양한 레스토랑
   - 추천: 쿠라마띠, 센타라그랜드, 노쿠 등

6. **🏄 액티비티 모험형** - 다양한 수상 스포츠와 액티비티
   - 추천: 오젠리저브볼리, 아난타라디구 등

7. **💕 허니문 로맨틱형** - 신혼여행 특화 로맨틱 성향
   - 추천: 콘래드, 세인트레지스, 반얀트리방갈리 등

8. **👨‍👩‍👧‍👦 가족 여행형** - 아이들과 함께하는 가족 여행
   - 추천: 쿠라마띠, 하드락, 센타라그랜드 등

## 📊 분석 항목 (6각형 그래프)

- **럭셔리**: 고급 시설과 서비스 선호도
- **수중환경**: 다이빙, 스노클링, 하우스 리프 관심도
- **라군**: 아름다운 바다 색깔과 사진 중요도
- **음식**: 다이닝과 올인클루시브 선호도
- **액티비티**: 활동적인 여행 선호도
- **가성비**: 예산 효율성 중요도

## 🔍 질문 구성

### 기본 정보
- 💰 예산 ($2,000 이하 ~ $6,000 이상)
- 📅 여행 기간 (3박 ~ 8박 이상)
- 👥 여행 동반자 (신혼/커플/가족/친구)

### 숙소 & 시설
- 🏠 워터빌라 vs 비치빌라
- 🏨 리조트 규모 (소형/중형/대형)
- 🏊 개인 풀빌라 필요성
- 🎨 인테리어 스타일
- 🔒 프라이버시 중요도

### 여행 스타일
- 🌊 수중환경 vs 라군 비율
- 🍽️ 음식 플랜 (BB/HB/FB/올인)
- 🍺 음주량
- 📸 사진/인스타 중요도
- 🎉 나이트라이프 관심도

### 액티비티
- ✈️ 이동 수단 (스피드보트/국내선/수상비행기)
- 🤿 리조트 내 vs 외부 투어
- 🎯 가장 중요한 우선순위
- 🌅 가장 하고 싶은 것
- 🌟 특별한 경험 (언더워터 다이닝/샌드뱅크 등)

## 🚀 시작하기

### 패키지 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000) 열기

### 정적 사이트 빌드

```bash
npm run build
```

빌드된 파일은 `out` 폴더에 생성됩니다.

> 커스텀 도메인에서 `/MaldivesMatch` 접두사가 필요 없으면  
> `NEXT_PUBLIC_BASE_PATH=""` 환경변수를 지정한 뒤 빌드하세요.

## 📦 GitHub Pages 배포

### 1. GitHub 저장소 생성 및 푸시

```bash
git init
git add .
git commit -m "Initial commit: Maldives Match Test"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/MaldivesMatch.git
git push -u origin main
```

### 2. GitHub Pages 설정

1. GitHub 저장소 → **Settings** 탭
2. 왼쪽 메뉴에서 **Pages** 선택
3. **Source**를 **GitHub Actions**로 선택
4. 자동으로 배포됩니다!

### 3. 접속

`https://YOUR_USERNAME.github.io/MaldivesMatch/`

## 🛠️ 기술 스택

- **Framework**: Next.js 14 (Static Export)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Charts**: Recharts (6각형 레이더 차트)
- **UI**: React
- **Deploy**: GitHub Pages
