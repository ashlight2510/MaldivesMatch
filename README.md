# 🏝️ 몰디브 매치 (Maldives Match)

나의 몰디브 여행 성향을 찾아주는 MBTI 스타일 테스트

## ✨ 기능

- 10개의 간단한 질문으로 여행 성향 분석
- 6가지 성향 타입 제공
- 6각형 레이더 차트로 시각화
- 성향에 맞는 리조트 추천

## 🎯 성향 타입

1. 👑 럭셔리 힐링형
2. 🐠 수중 탐험가형
3. 💙 라군 낭만형
4. 💰 가성비 실속형
5. 🍽️ 미식 올인형
6. 🏄 액티비티 모험형

## 🚀 시작하기

### 패키지 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000) 열기

### 정적 사이트 빌드

```bash
npm run build
```

빌드된 파일은 `out` 폴더에 생성됩니다.

## 📦 GitHub Pages 배포

### 1. GitHub 저장소 생성

```bash
git init
git add .
git commit -m "Initial commit: Maldives Match Test"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/MaldivesMatch.git
git push -u origin main
```

### 2. GitHub Pages 설정

1. GitHub 저장소 → **Settings** 탭
2. 왼쪽 메뉴에서 **Pages** 선택
3. **Source**를 **GitHub Actions**로 선택
4. 자동으로 배포됩니다!

### 3. 접속

`https://YOUR_USERNAME.github.io/MaldivesMatch/`

## 🛠️ 기술 스택

- **Framework**: Next.js 14 (Static Export)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Charts**: Recharts
- **UI**: React
- **Deploy**: GitHub Pages

## 📊 분석 기준

6가지 지표로 여행 성향을 분석합니다:

- 럭셔리: 고급 시설과 서비스 선호도
- 수중환경: 다이빙, 스노클링 관심도
- 라군: 아름다운 바다 색깔 중요도
- 음식: 다이닝과 음주 선호도
- 액티비티: 활동적인 여행 선호도
- 가성비: 예산 효율성 중요도
